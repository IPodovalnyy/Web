Содержание курса:

В курсе веб-разработки мы познакомим вас с клиентской веб-разработкой, которая включает в себя верстку - обучение HTML и CSS, и и написание кода на JavaScript. На курсе JavaScript будут рассотрены синтаксис, типы данных, работа с DOM и событиями, стандартная библиотека языка.

В курсе JavaScript также рассматривается большое количество библиотек и средст разработки - JQuery, JQuery UI, Underscore, Vue.js, Bootstrap, npm, Webpack, Less/Sass, Babel. Также обзорно расматриваем Grunt, RequireJS, TypeScript. В качествесерверной части используем Express.js. Обучение веб- разработке закрепляется на практике.

Вы научитесь создавать клиентскую часть сайта и получите большой кругозор по клиентским технологиям.
